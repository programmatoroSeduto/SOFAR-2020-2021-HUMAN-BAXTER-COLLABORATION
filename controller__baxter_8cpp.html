Description:
This node implements a server that receives as input which arm needs to move, which block and where it needs to put the block. It sets all the information and then it calculates the trajectory with moveit. The trajectory is divided in 5 phases: it first moves the gripper above the block, it moves down to pick the block, it moves above the block again, it moves on top of the goal and it goes down to the goal.
